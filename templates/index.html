Home

Welcome to my simple multipurpose python website made with Flask! Just a small project to practice my Python, enjoy!

Functionalities

So far the website has 3 functionalities:

  1. Projects visualiser
    • A simple table where you can display your current projects.
  2. Password generator
    • Lets you generate a customizable password.
  3. A file information visualiser
    • See all the information of the content of a text file you uploaded.

Technologies used

  • Flask
  • Bootstrap
